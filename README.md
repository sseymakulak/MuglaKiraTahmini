# MuglaKiraTahmini

# Kira Tahmini Uygulaması
Elde Edilen Gerçek Verilerle Makine Öğrenmesi Modelleri Oluşturarak Kira Tahmini Yapılması

## Proje Açıklaması
Bu proje, siteden elde edilen gerçek verilerle makine öğrenmesi modelleri oluşturarak kira tahmini yapılmaya çalışılmıştır. Veri seti üzerinde veri ön işleme işlemleri gerçekleştirildi, veri seti üzerinde istatistiksel analizler yapıldı ve makine öğrenmesi modelleri uygulandı.

## Kullanılan Kütüphaneler
- numpy
- pandas
- sklearn
